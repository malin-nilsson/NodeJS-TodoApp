# To Do App
To do app built with Node.js, Express.js and Handlebars üìù

This was a school assignment where the goal was to learn and develop skills in Node.js, Express.js and template engines.


![Screenshot](public/screenshot.png)


All tasks should have the following attributes:
- Id
- Created (date & time) 
- Description
- Done (true / false)

The objective was for users to be able to perform all CRUD operations on todos, display tasks depending on completed/not completed, and sort tasks based on when they were created, newest ‚Üí oldest and oldest ‚Üí newest.

## Installation
1. Clone the repo\
`git clone https://github.com/malin-nilsson/NodeJS-ToDoList`

2. Install npm packages\
`npm install`

3. Run Nodemon\
`nodemon`

Visit localhost:8000 to view the project
